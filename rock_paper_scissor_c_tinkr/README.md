# Rock Paper Scissors GUI

A simple Rock Paper Scissors game built with Python and Tkinter.

## Files

- `rock_paper_scissors.py`: The main game script with a graphical interface.
- `sandbox.py`: (existing workspace file)
- `send_tweet.py`: (existing workspace file)

## Requirements

- Python 3.x
- Tkinter (usually included with Python on Windows)

## Run the game

Open a terminal in the project folder and run:

```bash
python rock_paper_scissors.py
```

## How to play

- Click one of the buttons: `Rock`, `Paper`, or `Scissors`.
- The computer will randomly choose its move.
- The result and current score are shown in the window.
- Click `Quit` to close the game.

## Notes

- The game keeps a running score for the player and the computer.
- Ties do not change the score.
